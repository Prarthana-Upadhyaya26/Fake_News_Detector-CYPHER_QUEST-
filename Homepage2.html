LLM

A Large Language Model (LLM) is an advanced AI that understands and generates human-like text using a deep neural network trained on vast text data. It performs tasks like translation, summarization, text generation, and answering questions by leveraging patterns in the data.

NLP

Natural Language Processing (NLP) is an AI field focused on the interaction between computers and human language. It performs tasks such as text classification, sentiment analysis, and machine translation. Other key tasks include named entity recognition, speech recognition.

Sentiment Analysis

Sentiment analysis, a subfield of NLP, determines the emotional tone of text, classifying it as positive, negative, or neutral. It helps understand the author's attitude and emotions. It can help suppress negative reviews, trolls and bad comments.
